# DMZ 다이어리 — UI 맵

피터공↔아리공↔아리온↔백도 에이전트가 같은 언어로 소통하기 위한 네이밍 규칙.

## 화면 (Screen)

| 이름 | 코드 ID | 설명 |
|------|---------|------|
| **로그인** | `#login-screen` | 타이틀 + 이름/코드 입력 |
| **선택** | `#select-screen` | 캐릭터(시선) 선택 |
| **게임** | `#game-screen` | 자료 카드 + 빈칸 채우기 |
| **전달** | `#delivery-screen` | 두루미 애니메이션 + 기록 전송 |
| **결과** | `#result-screen` | 점수 + 요약 |

## 주요 UI 요소

| 이름 | 코드 클래스 | 위치 | 설명 |
|------|-----------|------|------|
| **윈도우바** | `.win-bar` | 로그인 | 상단 검정 줄무늬 바 ("DMZ 다이어리") |
| **타이틀** | `.login-title` | 로그인 | "DMZ 다이어리" 큰 글씨 |
| **서브타이틀** | `.login-sub` | 로그인 | "아키비스트가 DMZ를 기록하는 방법" |
| **힌트** | `.login-hint` | 로그인 | 하단 안내 텍스트 |
| **캐릭터카드** | `.char-card` | 선택 | 시선 선택 카드 (이름+설명+시대) |
| **헤더바** | `.game-header` | 게임 | 상단 고정 바 (장소+진행률) |
| **자료카드** | `.source-card` | 게임 | 편지/신문/사진/구술 카드 |
| **자료제목** | `.card-title` | 게임 | 카드의 굵은 제목 |
| **자료설명** | `.card-sub` | 게임 | 카드 아래 파란 설명 |
| **상태뱃지** | `.card-status` | 게임 | "미해독" / "해독" 표시 |
| **자료뷰어** | `.source-detail` | 게임 | 카드 클릭 시 전체화면 상세보기 |
| **빈칸** | `.blank-slot` | 자료뷰어 | 클릭하면 답 입력하는 빈칸 |
| **답변모달** | `.answer-modal` | 게임 | 하단에서 올라오는 답 입력 창 |
| **페이즈배너** | `.phase-banner` | 게임 | "PHASE 1" 등 단계 표시 |
| **토스트** | `.toast` | 전체 | 정답/오답 팝업 |

## 자료 유형 (자료뷰어 안)

| 이름 | 코드 클래스 | 설명 |
|------|-----------|------|
| **편지** | `.letter-paper` | 세리프체, 서명 포함 |
| **일기** | `.diary-paper` | 이탤릭, 줄노트 배경 |
| **학술문서** | `.scholar-paper` | 세리프체, 두꺼운 왼쪽 보더 |
| **신문기사** | `.newspaper-paper` | 신문 헤더+제목+본문 |
| **사진** | `.photo-frame` | 체커보드 플레이스홀더+캡션 |
| **구술녹음** | `.oral-player` | 카세트 아이콘+대사 |

## 소통 예시

- "**선택** 화면의 **캐릭터카드** 글씨가 너무 커"
- "**게임** 화면의 **자료설명** 크기 좋아"
- "**답변모달**의 버튼이 작아"
- "**편지** 자료의 **빈칸** 색상 변경"
