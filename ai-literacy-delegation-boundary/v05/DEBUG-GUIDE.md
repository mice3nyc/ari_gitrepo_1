#### 검증 1 — Final Report 25 케이스 (4유형×5등급, 약 10분)

##### 무엇을 보는가

학기 끝났을 때 뜨는 **결과 화면 6컷 카툰 + 시상 + 무드 + 4유형 텍스트**가 25개 조합 모두에서 정상 렌더되는지.

| 4유형 (위/도) | 5등급 |
|---|---|
| pp (위+ 도+) | S / A / B / C / D |
| pn (위+ 도−) | S / A / B / C / D |
| np (위− 도+) | S / A / B / C / D |
| nn (위− 도−) | S / A / B / C / D |
| mid (0,0) | S / A / B / C / D |

##### 절차 (한 케이스당 30초)

1. 디버그 패널 **위/도 4유형 (강제)** 섹션에서 한 유형 클릭 (예: `위+ 도+ (pp)`)
2. **Final Report (5 시나리오 모두 등급 X)** 섹션에서 등급 한 개 클릭 (예: `S`)
3. 결과 화면이 뜸 → 아래 **체크 5개** 확인
4. 화면 아무 데나 클릭으로 닫고, 다음 조합으로

##### 체크 5개 (한 케이스마다)

- [ ] **상단 4박스**: 총점 / Lv / 위 / 도 숫자 정상 표시 (위/도는 강제로 세팅한 값)
- [ ] **위/도 4유형 텍스트 박스**: pp/pn/np/nn/mid 중 강제 세팅한 유형 텍스트 일치 (4유형은 §10.13 코드 참조)
- [ ] **6컷 카툰**: 시나리오 5개 × 6컷 = 30컷이 좌→우 strip으로 렌더. 빈 칸/깨진 이미지 없음
- [ ] **시상 이미지** (각 시나리오 6번째 컷): S=초록 / A=초록 / B=검정 / C=주황 / D=빨강 색상 일치
- [ ] **무드 대사**: S/A/B/C/D 등급에 맞는 대사 한 줄 정상 표시 (예: S=`와, 이번 학기 진짜 멋졌어. AI랑 베프가 됐네.`)

##### 결과 기록

5×5 표를 화면에 띄워놓고 ✅/❌ 체크하면서 진행. ❌ 발견 시 어떤 조합·뭐가 이상한지 한 줄 기록 → 아리공한테 넘겨주시면 됨.

#### 검증 3 — 손계산 (5분, 비용 표시 검증)

##### 공식 (지금 박혀 있는 것)

표시 비용 = **max(0, round(raw비용 × 0.6) − 역량점 × 2)**

- 시간 비용은 **위(위임 선택력)**가 깎음. 도(도메인 지식)는 영향 X
- 에너지 비용은 **도**가 깎음. 위는 영향 X
- `0.6`은 `CONFIG.resourceCostMultiplier` (글로벌 비용 배율)
- `2`는 `CONFIG.competencyDiscountMult` (역량 배율 — 4/30 8.12에서 3→2 완화)

##### 절차

1. 디버그 패널 **위/도 4유형** → `mid (0,0)` 클릭 (위=0, 도=0 깨끗한 시작)
2. 시나리오 처음으로 돌아가기: **Reset** 클릭
3. 일반 플레이로 첫 시나리오(자기소개) 진입 → tier1/tier2 선택 화면에서 **4박스 비용 표시** 확인
4. 박스에 보이는 숫자 = `round(raw × 0.6)` (할인 0). 예: raw 30 → 18 표시
5. 디버그 패널 **위 +1 / 위 -1** 버튼으로 +3 만들고 같은 화면 재확인 → 시간 비용 −6 (배율 2 × 위 3) 적용 표시
6. **위 -3, 도 -3**으로 만들면 시간/에너지 비용 +6씩 페널티 표시

##### 체크
- [ ] 위 0, 도 0: 박스 숫자 = round(raw × 0.6) 일치
- [ ] 위 +3: 시간 비용이 정확히 6 줄어듦
- [ ] 위 -3: 시간 비용이 정확히 6 늘어남 (페널티)
- [ ] 도 +3: 에너지 비용 6 줄어듦
- [ ] 도 -3: 에너지 비용 6 늘어남

#### 검증 4 — 발란스 진단 (피터공 4/30 발견 후속)

> 가장 비싼 선택만 해도 학기 끝까지 갈 수 있음 → 챌린지 부족.

CSV 4종 (별도 작업으로 추출 중)이 나오면 엑셀에서 다음 3개를 본다.

- 시나리오별 **최비싼 경로 합** vs **현재 자원 max(시간 100, 에너지 100)**
- 학기 5개 합산 시 **최비싼 경로**라도 자원이 0 floor 안 찍는지
- 레벨업 임계값 `[0, 20, 40, 65, 100]`이 너무 빨라서 후반 메터 max 늘어나며 또 자원 늘어나는지

CSV 도착하면 다시 합쳐서 봄.
