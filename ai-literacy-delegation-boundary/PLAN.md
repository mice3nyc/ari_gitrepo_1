# AI 리터러시: 위임의 경계 — Playable Prototype v0.1

**최종 업데이트**: 2026-04-25

## 목적

경기도교육청 하이러닝 탑재용 놀공 AI 리터러시 몰입형 콘텐츠의 1차 Playable Prototype.
학생이 AI에게 어디까지 맡길지 판단하는 선택 기반 학습 경험.

> 지금 이 상황에서, 어디까지 AI에게 맡겨도 되고 무엇은 내가 해야 하는가?

## 대상

중1~고1 학생. 수행평가, 발표, 조별과제, 시험 공부, 진로 탐색에서 AI 위임 판단을 경험.

## 핵심 경험

"AI를 쓸까 말까"가 아니라 "어디까지 맡길까"를 고민하게 만드는 것.

## 핵심 루프

상황 제시 → 판단 관점 확인 → 위임의 경계 선택 → 자원 변화(시간/에너지/경험) → 스킬 성장 → 피드백 → 다음 시나리오 → 최종 리포트

## 판단축 (3개)

| 축 | 학생용 질문 |
|----|-----------|
| 내 생각 | AI를 쓰기 전에, 내 생각이 먼저 있었나? |
| 맡길 일 | 이 일은 AI에게 맡겨도 되는 일인가, 내가 해야 하는 일인가? |
| 다시 보기 | AI가 준 답을, 나는 다시 볼 수 있나? |

## 자원값 (3개)

- **시간**: 초기 60. AI 사용 시 절약, 직접 할수록 소모
- **에너지**: 초기 50. 사고/검토에 소모
- **경험**: 누적. 스킬 레벨업 연동

## 시나리오 구성 (6개)

| # | Act | 제목 | 장소 |
|---|-----|------|------|
| 1 | 1 | 발표 자료가 너무 많다 | 학교 도서관 |
| 2 | 1 | AI가 고친 문장이 훨씬 좋아 보인다 | 집 책상 |
| 3 | 2 | 조별과제에서 친구가 AI로 다 만들자고 한다 | 교실 |
| 4 | 2 | AI 요약만 보면 충분할 것 같다 | 집/독서실 |
| 5 | 2 | AI가 알려준 정보가 맞는지 모르겠다 | 컴퓨터실 |
| 6 | 3 | AI가 내 진로를 너무 잘 아는 것 같다 | 집/상담실 |

## 이번 버전에서 하지 않는 것

- 서버, 로그인, 계정
- 실제 AI API 호출
- 교사 대시보드
- 복잡한 애니메이션
- 이미지 생성
- 자원 고갈 시 제약 (플테 후 결정)

## 이후 확장 가능성

- 서버 전송 (trackEvent에 fetch 추가)
- 교사 대시보드
- 추가 시나리오/챕터
- 실제 이미지
- 자원 고갈 제약 메카닉
- 초등 고학년 버전
