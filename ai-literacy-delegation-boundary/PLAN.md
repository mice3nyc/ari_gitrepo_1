## 목적

경기도교육청 하이러닝 탑재용 놀공 AI 리터러시 몰입형 콘텐츠.
학생이 AI에게 어디까지 맡길지 판단하는 선택 기반 학습 경험.

> 지금 이 상황에서, 어디까지 AI에게 맡겨도 되고 무엇은 내가 해야 하는가?

## 대상

중1~고1 학생. 수행평가, 발표, 조별과제, 시험 공부 등 학생 일상에서 AI 위임 판단을 경험.

## 핵심 경험

"AI를 쓸까 말까"가 아니라 **"어디까지 맡길까 + 결과를 어떻게 검토할까"**를 고민하게 만드는 것.

학생은 결과를 거슬러 자기 선택을 추적하면서 **"내가 이렇게 빌드했구나"**를 깨닫는다 (인과 자각, 4/27 탐구 8 통찰).

## 두 역량 (v0.3, 회의 합의)

기존 v0.2의 판단축 3개(내 생각/맡길 일/다시 보기) → **두 역량으로 통합**:

| 역량 | 게임 단계에서 작동 |
|---|---|
| **위임 선택력** | 1차·2차 선택 |
| **지식 (도메인)** | 검토 단계 |
| (최종) | 두 역량 합산 → 점수 + 아이템 |

## 게임 트리 구조 (v0.3)

```
상황 (1)
  ↓ — 위임 선택력 1
1차 선택지 (3)         A 직접 / B 위임 / C 우회
  ↓ — 위임 선택력 2
2차 선택지 (각 3 = 9)
  ↓
결과 (9, raw 결과물)
  ↓ — 지식 역량
검토 (각 3 = 27)        R1 그대로 / R2 한 번 읽기 / R3 크로스체크
  ↓
최종 점수 + 아이템 (27)
```

총 76노드 / 시나리오. 학생 1회 플레이 = 3선택(1차→2차→검토)으로 1경로 = 27분기 중 1.

## 시나리오 5개 (v0.3, 항목별)

| # | 항목 분류 | 1차 prototype | 시간 |
|---|---|---|---|
| 1 | 자료 조사 / 정보 검색 | 달 착륙 보고서 | 8분 |
| 2 | 글쓰기 / 의견 형성 | 학교 신문 자판기 사설 | 8분 |
| 3 | 본인 경험 기반 | **어린왕자 독후감 (모델)** | 12분 |
| 4 | 분석 / 판단 | 친구 다툼 중재 | 8분 |
| 5 | 윤리 / 개인정보 | 친구 비밀 AI 입력 | 7분 |

전체 플레이 = 50분 (인트로 3분 + 시나리오 5개 × 8~12분 + 마무리 4분).

## 시나리오 풀 확장 (라이키 형식 한 줄 풀)

각 항목당 5~6개 한 줄 후보 보유 (시나리오 작성 계획 노트 §12). prototype 검증 후 다른 후보로 변형 가능.

## 점수 + 아이템 (5등급)

| 등급 | 점수 | 아이템 |
|---|---|---|
| S | 90~100 | 「깊은 독서가」급 |
| A | 70~89 | 「검증의 눈」급 |
| B | 50~69 | 「반쪽 독후감」급 |
| C | 30~49 | 「위태로운 위임」급 |
| D | 0~29 | 아이템 없음 |

(시나리오별 아이템 이름 톤 다름)

## 외부 자원 — 학맞통 카드 (70장)

7주제 × 10장. Pair 자음 빈칸 형식 (학맞통 엑셀 차용).

| 주제 |
|---|
| AI 환각 식별 |
| 검증 방법 |
| 개인정보 |
| 위임 판단 |
| 사용법 |
| AI 한계 |
| 비판적 사고 |

**결합**: 시나리오 진행 중 카드 보유 → 검토 단계에서 발견 폭 ↑ → 점수 +20.

## 6컷 화면 (시나리오당)

| 컷 | 화면 |
|---|---|
| 1 | 상황 제시 |
| 2 | 1차 선택지 (A·B·C) |
| 3 | 2차 선택지 (선택된 1차의 3개) |
| 4 | 결과 (raw 결과물 텍스트) |
| 5 | 검토 선택 (R1·R2·R3) — **인과 자각의 핵심** |
| 6 | 최종 점수 + 획득 아이템 |

기존 v0.2 = 3컷(s01_ch1·ch2·ch3). v0.3 = 6컷 ×5시나리오 = 30컷 이미지 작업 필요.
