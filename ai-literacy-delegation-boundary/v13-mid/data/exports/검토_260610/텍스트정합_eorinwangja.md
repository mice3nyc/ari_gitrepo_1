#### 검사 범위 및 방법

27개 결말(A1R1~C3R3) 전수 확인. 검사 필드: awareness, cartoonCaption1~5, cut6Feedback, shortFeedback, reportFeedback, reportReflection, reportPathSummary, replaySuggestion, results 섹션, reviewSupplements, reportCardSummary.

경로 의미 기준:
- tier1 A = 책을 직접 읽기 / B = AI·외부 도움으로 실마리(책 안 장면은 일부 직접 확인) / C = 책 읽기 건너뜀
- tier2 B3 = 책을 읽으면서 AI와 궁금한 점 나누기 (B1·B2와 구분: B1은 요약+한 장면, B2는 유명 문장 앞뒤)
- R1 = 바로 제출 / R2 = 한 번 확인 / R3 = 깊게 확인

가장 중요한 발견: **B3 경로 전체에서 tier2 라벨("책을 읽으며 AI에게 궁금한 점을 물어보고 내 생각을 정리")이 실제 텍스트 필드들(results.B3, cartoonCaption3, reviewSupplements.B3R1)과 일관되게 어긋난다.** 세 개의 독립 필드 모두에서 B1/B2 행동이 서술되어 있어, 텍스트 생성 시 B3 경로 정의가 혼동되었을 가능성이 높다.
