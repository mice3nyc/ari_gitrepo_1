##### B. ★ 모바일 전용 문제 그대로 (DN 메모와 직결)

- viewport: `user-scalable=no`, 핀치줌 막힘
- 전체 레이아웃 `max-width: 400~500px` 컬럼 1개 (대형 디스플레이에서 가운데에 좁게 뜸)
- **공간 설치물용 빌드는 3곳만 손대면 됨**: viewport meta / max-width 값들 / 폰트 크기
