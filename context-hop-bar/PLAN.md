# Context Hop Bar — 개발 계획

> 피터공의 Context Hopping을 실시간으로 시각화하는 모니터링 바.
> "지금 내가 어디에 있고, 얼마나 자주 옮겨 다녔는지를 한눈에."

## 프로젝트 개요

- **이름**: Context Hop Bar
- **컨셉**: 터미널 옆에 띄워놓는 슬림한 웹 바. Claude Code 작업 중 어떤 프로젝트를 건드리는지 실시간 표시
- **영감**: 세션146 터미널 분석 — 31항목, 30hop. "의도한 hop" vs "끌려가는 hop" 구분
- **기술 원칙**: 단일 HTML, 바닐라 JS, 프레임워크 없음
- **디자인**: POCO와 동일 — Minimal Graphical, 북유럽/독일 감성
- **크기**: 폭 넓고 높이 낮음 (터미널 옆 또는 위에 배치). 기본 800x120px, 리사이즈 가능

---

## 아키텍처

```
┌─────────────────────────────────────────────┐
│ Claude Code (터미널)                         │
│  → tool call 발생                           │
│  → Hook 실행 (settings.json)                │
│  → /tmp/context_hop_log.jsonl에 한 줄 추가   │
└──────────────────┬──────────────────────────┘
                   │ 파일 시스템
┌──────────────────▼──────────────────────────┐
│ Python 미니 서버 (python3 -m http.server)    │
│  → /tmp/context_hop_log.jsonl 서빙           │
│  → index.html 서빙                          │
└──────────────────┬──────────────────────────┘
                   │ HTTP
┌──────────────────▼──────────────────────────┐
│ 브라우저 (index.html)                        │
│  → 2초마다 fetch → 로그 파싱                 │
│  → 프로젝트 블록 하이라이트/fade             │
│  → hop 카운터 + 타임라인                     │
└─────────────────────────────────────────────┘
```

### 레이아웃

```
┌────────────────────────────────────────────────────────────┐
│ ● Context Hop Bar              hops: 12  focus: 3m  14:30 │
├────────────────────────────────────────────────────────────┤
│ [DN] [시스템] [휴룹] [글쓰기] [개발] [노트] [메일] [텔레] │
│  ░░    ░░     ██     ░░      ░░     ░░    ░░    ░░       │
├────────────────────────────────────────────────────────────┤
│ ·  · ··  ·    ···  · ·  ·· ·    · · ·    ·   ·· ·  ·    │
│ 09:00          12:00          15:00          18:00         │
└────────────────────────────────────────────────────────────┘
```

- 상단: 제목 + hop 카운터 + 현재 focus 시간 + 시계
- 중단: 프로젝트 블록 (██ = 현재, ░░ = 비활성, opacity로 최근도 표시)
- 하단: 타임라인 (dot = hop 발생 시점)
