## 현재 단계
Canon 파일 4종 초안 완료. 피터공 검토 대기 중.

## 결정된 것
- 수집 단위: D(신호) 기반 + C(검증) 관문
- 4가지 신호: 🔵기술 / 🟡사회 / 🔴개인 / 🟣미래
- 역할: 시그널 에이전트
- 완성물: 미래 브리핑 (학생마다 다름 = 거울)
- 거울 원칙: "수집한 것이 나를 비춘다"
- 검증: 노이즈 필터링 (관문이지 목적 아님)
- 개인/미래 신호: 글쓰기 아님, 조립식(조각 선택)

## 미결정
- UI: 대시보드? 피드? 카드 테이블? → 프로토타입 루프에서 실험
- 에피소드 1의 구체적 정보 카드 내용 → 기획 루프에서 작성
- 교사 가이드 구조 → 프로토타입 후
