## 병목/이슈

1. **node 검증 불가**: Bash에서 node 실행이 차단됨. JS 문법 검증은 수동 코드 리뷰로 대체. 브라우저에서 열어 확인 필요
2. **카드 11번(은서 인터뷰)의 활용도**: `assignableTo: ['personal']`이지만, 메인 게임에서 기술/사회만 등록. 이 카드는 개인 신호 배경 읽기와 중복 역할. 카드 브라우저에서 읽기만 하고 등록은 못함 → 의도된 설계(읽기 = 이해, 등록 = 판단 분리)이지만 학생이 혼란할 수 있음
3. **모바일 보드 컴팩트 모드**: 4개 슬롯을 가로 일렬로 → 텍스트가 매우 작아질 수 있음. 실기기 테스트 필요
4. **15~20분 타이밍**: 12장 카드 읽기 + 등록 + 개인/미래 선택 + 조립 + 브리핑 + 거울. 예상 시간 ~12~15분. 빠를 수 있음
