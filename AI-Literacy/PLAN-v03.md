## 피터공 피드백 R1 (4/3)

### FB-1: 선택지가 카드 느낌이 아니라 버튼
**현재**: 선택지가 버튼처럼 보임
**요청**: 카드 느낌. 스와이프해서 위/좌/우로 밀면 선택되는 구조
**방향**: 선택지 자체가 카드형 UI — 둥근 모서리, 그림자, 패딩. 좌우 스와이프로 선택

### FB-2: 아이콘이 너무 작고 게임 느낌 부족
**현재**: ◎◈◇ 텍스트 심볼, 작음
**요청**: 오리지널 Mac OS 미니멀 스타일. 아이콘도 그 느낌. UI 전체가 선명하게 게임처럼
**방향**: 흑백 매킨토시 픽셀 아트 느낌 — 1bit 아이콘, 선명한 보더, Chicago/Geneva 같은 비트맵 폰트 느낌. Susan Kare 스타일 아이콘을 SVG 인라인으로

### FB-3: 3지선다 카드가 다 겹쳐있다
**현재**: translateY 오프셋 8px/16px — 너무 좁아서 카드 구분 불가
**요청**: 카드가 구분되게 간격 벌리기
**방향**: 세로 간격 넓히기. 각 카드의 텍스트가 보여야 함

### FB-4: 카드 위에 스탯 프리뷰 제거
**현재**: hover/드래그 시 "중심↑ 융합↓" 같은 스탯 변동이 카드에 미리 표시됨
**요청**: 제거. 플레이어가 선택의 결과를 느껴야지 미리 보여주면 안 됨
**방향**: choice-preview 텍스트 완전 제거. hover 시 스탯 바 하이라이트 제거. 선택 후에만 변동 표시
