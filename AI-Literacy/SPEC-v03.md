## v0.3 변경 사항 (5건)

### 1. 카드 스와이프 선택 (레이아웃 변경)

**현재**: 선택지가 세로 버튼 리스트
**변경**: Reigns 스타일 — 중앙에 상황 카드 1장, **좌우 스와이프** (또는 좌우 버튼 탭)로 선택

구현 방식:
- 카드 중앙 배치, 터치/드래그로 좌우 기울어짐
- 좌 = 선택지 A, 우 = 선택지 B (2지선다 기본)
- 3지선다인 경우: **카드 3장 스택** — 위아래로 겹치듯 쌓여 있고, 탭/스와이프로 선택
- 카드 기울어질 때 해당 선택의 스탯 프리뷰 표시
- 놓으면 스와이프 방향으로 카드가 날아가며 결과 전환

**제약**: 바닐라 JS만. 라이브러리 없음. touch + mouse 이벤트 직접 구현.

### 2. 역량 아이콘 추가

**현재**: 텍스트만 (중심잡기 / 융합하기 / 성찰하기)
**변경**: 각 역량 옆에 아이콘 추가

후보 (시스템 이모지 — 폰트 의존 없음):
- 중심잡기: ⚓ (닻 — 중심을 잡는다)
- 융합하기: 🔗 (연결 — 도구와 나를 잇는다)
- 성찰하기: 🪞 (거울 — 자신을 돌아본다)

> ~~결정: 텍스트 심볼~~ → **R1 변경: 오리지널 Mac OS 미니멀 스타일**. 1bit 흑백 픽셀 아트 아이콘 (Susan Kare 느낌). SVG 인라인. 크기 키움

### 3. 스탯 변동 시 원인 텍스트 표시

**현재**: 숫자 +/-가 팝업됐다 사라짐 (1.2초)
**변경**: 숫자와 함께 **무엇이 영향을 줬는지** 텍스트 표시

예시: 선택지 "그대로 제출한다" 선택 시
- 중심잡기 옆: <span style="color:red">-2 "내 판단 없이"</span> (빨강, 페이드아웃)
- 성찰하기 옆: <span style="color:red">-2 "확인 안 함"</span> (빨강, 페이드아웃)

구현:
- 각 선택지 stats 배열에 `reasons` 필드 추가: `reasons: ["내 판단 없이", "", "확인 안 함"]`
- changePopup 애니메이션에 텍스트 포함
- +는 초록(#1a8c1a), -는 빨강(#c44)
- 1.5초 표시 후 페이드아웃

### 5. 시나리오 연속 진행 (막/레벨 구조)

**현재**: 시나리오 선택 화면 → 1개 플레이 → 프로필 → "다른 시나리오"
**변경**: 시나리오가 **순서대로 자동 진행** — 막(Act) 구조

```
Act 1: 숙제 + ChatGPT (4라운드)
  → 중간 프로필 스냅샷
  → "다음 상황으로" 전환 카드
Act 2: 독후감 "모르겠다" (4라운드)
  → 중간 프로필 스냅샷
  → "다음 상황으로"
Act 3: 미술 AI 그림 (4라운드)
  → 최종 프로필 (누적 스탯)
```

변경 포인트:
- 시나리오 선택 화면 제거
- 스탯이 Act 간 **누적** (리셋 없음)
- 각 Act 사이에 전환 카드: "Act 2 — 독후감" (연극 막 느낌)
- 라운드 인디케이터: Act 내 진행 + 전체 Act 진행 표시
- 최종 프로필은 3개 Act 합산

> **결정: (a) 누적** — 최종 유형 나옴. **(b) 고정** — 숙제→독후감→미술. (c) 전환 카드로 처리
