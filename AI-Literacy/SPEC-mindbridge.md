# AI 리터러시 — 마인드 브릿지 프로토타입 SPEC

## 개요
- **프로젝트**: 경기도교육청 하이러닝 AI 리터러시 콘텐츠
- **교육 목표**: AI로 해도 되는 것 / 하면 안 되는 것을 구분하는 위임 판단력
- **포맷**: 연쇄 분기 시나리오 + 역량 스탯 시스템
- **레퍼런스**: Reigns (Devolver Digital) — 카드 선택 + 즉각 스탯 변동
- **기술 스택**: 단일 HTML, 바닐라 JS, 프레임워크 없음

## 스탯 시스템
- **표면 (학생이 보는 것)**: 중심잡기 / 융합하기 / 성찰하기 (김현철 3축)
- **내부 로직**: 12역량이 복합 작동 → 3축으로 합산
- **핵심 원칙**: 망하는 선택 없음. 트레이드오프만 있음.

## 시나리오 구조
- ABC(세 컷) 모델 폐기 → 연쇄 분기 4라운드 (8-12컷)
- 매 라운드: 상황 → 선택(2-3개) → 결과 + 스탯 변동 → 다음 상황
- 최종: 프로필 표시 + 성찰 코멘트
- 재플레이: 다른 경로 체험 가능

## 시나리오 목록 (v0.2)
1. **숙제 + ChatGPT** — hallucination 분기 포함
2. **독후감 "모르겠다"** — 혼란이 사라지는 문제
3. **미술 AI 그림** — 소유권/솔직함
