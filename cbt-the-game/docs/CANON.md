## 덱빌딩 CBT 게임 — 페이퍼 프로토타입 캐넌

> 이 노트가 디지털 재구현의 단일 기준점이다. 출처: `Assets/incoming/CBT Game/게임 과몰입 덱빌딩게임 제작 과정 누적 기록.pdf`(52p, 1~16차 빌드, 2024.08~2025.03). 페이지 번호는 그 문서 기준.
> **무시할 것**: 임상 빌드 `dev.mindshiftdtx.com`(Next.js, 삼성병원 임상 DTx)와 `250911 CBT게임 기획 문서_최종`(임상용 1:1 단순화). 별개 트랙이며 캐넌이 아니다.

### 한 줄

도미니언식 덱빌딩으로 "충동 → 생각으로 억제 → 생각이 모여 실천 → 충동·과몰입 제거"를 직접 플레이로 겪게 하는 게임. 설명이 아니라 규칙이 CBT를 시연한다. 성대·놀공 플레이테스트에서 효과를 확인했다.

### 시작 덱

- **충동카드 7장** — 플레이어가 게임 시작 전 자신이 경험한 충동 7가지를 직접 선택한다("자신이 경험한 충동 선택", p.30 7차).
- **생각하는힘 카드 3장** — 도미니언의 "돈 카드" 역할. 생각카드·실천카드를 사는 자원(p.1).

### 한 턴 = 하루

1. 카드가 1장씩 자동으로 뽑힌다. **하루 최대 5장, 원할 때 멈출 수 있다.** 뽑기를 멈추는 것 자체가 충동 조절 메카닉이다.
2. 뽑은 충동을 확인한다. 내가 어떤 충동을 느끼는지 본다.
3. **생각하는힘으로 같은 유형의 생각카드를 구매**하면 그 유형 충동이 억제되어 버린덱으로 간다. 억제 못 한 충동은 **과몰입행동 칸**으로 이동한다.
4. 과몰입행동 칸으로 간 충동을 다음날도 억제하지 못하면 그 즉시 과몰입행동이 발동한다(충동이 늘거나 실천을 방해).
5. **같은 유형 생각카드 2장이 모이면 실천카드가 나타난다.** 실천카드는 해당 유형의 충동·과몰입행동을 영구 제거한다("생각이 모여 실천으로 이어진다").

### 승/패 (디지털은 9차 "칸 채우기" 채택)

- **클리어**: 실천 6칸이 모두 실천카드로 채워짐 (9차 빌드, p.36 근방).
- **실패**: 과몰입행동 6칸이 모두 참.
- (16차 점수 시스템 — 실천하는힘 10점 클리어 / 함정카드 5개 실패 — 은 디지털 1단계에서 채택하지 않음.)

### 충동 유형 (K-IGD 6가지 — 최종)

7차에서 IGD 진단 기준 K-IGD 6항목으로 정착: **현저성·기분변화·내성·금단·갈등·재발**. 보드는 이 6유형에 맞춰 6칸 × 3영역(과몰입행동/실천/생각).

> 유형 진화: 9가지(3차) → 4가지(5차) → 통제력상실·내성·금단 3×3=9 재확장 → K-IGD 6(7차 최종). 생각카드·실천카드 유형은 각각 충동·과몰입행동 유형과 동일(같은 type 매칭).

### 왜 효과를 봤나

- "충동을 계속 억제하지 못해 과몰입행동이 추가되는 것이 위기감으로 다가왔는데, 실천카드로 그게 저지되는 걸 보는 게 매우 안정감·만족감을 줬다"(p.11 4차 핵심 피드백).
- "어떤 생각을 해야 이 충동을 억제할 수 있을지 고민하며 생각카드·실천카드 내용을 자연스럽게 읽게 되었다."
- 성대 테스트(10차): "[생각]으로 [충동]을 억제하는 경험이 게임 과몰입을 치료할 효과적 경험이라 생각했다. 생각카드를 결정할 때마다 상당히 몰입됐다." 학생들이 게임 중 "내가 과몰입이구나"를 자각.

### 디지털 재구현에서 정할 것 (문서가 남긴 미결)

- **1:1 유형 매칭 엄격도**: 16차 — "같은 유형 안에 있다고 정답 인정하면 게임이 많이 망가짐. 1:1로 매칭돼야 유의미"(p.후반). → 디지털도 유형 1:1 매칭 유지.
- **유형화 존치**: 13~16차 최대 미해결. 의학 소견·콘텐츠화 측면에서 유형을 얼마나 유지할지. 1단계는 6유형=6칸으로 진행하되 재검토 여지.
- **카드를 읽게 만드는 강제**: 반복 피드백 — 실천·과몰입 카드 텍스트를 안 읽게 됨. 구매 행동을 넣으니 잘 읽게 됨(p.31 9차). 충동 유형을 가리니 내용을 읽게 됨(p.31). → 디지털: 유형 미공개 + 구매 인터랙션.
- **연출**: 뽑기 멈춤, 과몰입 칸 가득참=게임오버("돌이킬 수 없는 상태"), 유형 공개 타이밍(앞/뒷면).
- **매일 반복 동기**: 매일 플레이 시 누적 기록 화면(p.15 5차 아이디어). 디지털화의 핵심 부가가치 → 2단계.

### 카드 콘텐츠 출처 (게이트)

페이퍼 프로토 전용 "게임과몰입 덱빌딩게임 콘텐츠 시트"는 **현재 볼트에 없다**. 캐넌 docx에는 6유형 분류 + 실제 카드 텍스트 예시(아래)가 있다. 1단계는 이 예시로 대표 카드 세트를 작성하고(placeholder), 실제 시트를 피터공에게 받아 교체한다.

**충동카드 예시(1차 원문)**: "롤 ㄲㄲ 얼른 들어와" / "이길때까지 계속 한다" / "오늘은 승급한다" / "게임을 멈출 수 없어" / "밤새 달린다" / "집에 왔다. 고로 난 게임을 켠다" / "내가 캐리함"
**생각카드 예시**: LV1 "지금 몇시지?" / "나 게임 몇시간 했지?" / "나 밥은 먹었나?" — LV2 "게임을 너무 많이 했나?" / "지고 있는데 왜 계속 하고있을까?" — LV3 "내가 게임말고 좋아하던 일이 뭐였더라?"
**실천카드 예시(효과 포함)**: "스트레칭을 하자"(생각카드 +1 간주) / "오늘은 게임 그만하자"(패의 게임카드 1장 폐기) / "자기 전까지 다른 취미생활 하기"(생각카드 +2 간주)
**과몰입행동 유형(4차)**: 절제력 감소 / 불규칙한 생활패턴 / 스트레스 / 공허함 → 대응 실천: 절제력 상승 / 규칙적 생활 / 스트레스 감소 / 공허함 감소
