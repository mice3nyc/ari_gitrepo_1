# POCO Map — 디자인 가이드

> Minimal Graphical. 북유럽/독일 감성.
> "박물관의 정보 디자인처럼 — 조용하지만 정확하다"

## 디자인 원칙

1. **여백이 정보다** — 빈 공간이 시선을 안내한다
2. **흑백 기본, 컬러는 의미가 있을 때만** — 장식이 아닌 데이터를 위한 색
3. **타이포가 UI다** — 버튼/아이콘 최소화, 텍스트가 인터페이스
4. **한눈에 읽힌다** — 2D 평면, 스크롤 아닌 한 화면

## 색상 팔레트

```
배경:      #FAFAF8  (따뜻한 화이트, 종이 느낌)
텍스트:    #1A1A1A  (거의 블랙)
서브텍스트: #8C8C8C  (중간 그레이)
라인/보더: #E0E0E0  (연한 그레이)

데이터 색상 (취득 방법별):
  구매:     #2D2D2D  (다크 그레이)
  기증:     #6B7B8D  (스틸 블루)
  발굴:     #8B7355  (번트 엄버)
  약탈/분쟁: #C44D3F  (테라코타 레드 — 유일한 강한 색)
  미상:     #B8B8B8  (라이트 그레이)

악센트:    #C44D3F  (dispute 하이라이트 전용)
```

## 타이포그래피

```
제목:    'Inter', sans-serif — 400/600 weight
본문:    'Inter', sans-serif — 400 weight
숫자:    'JetBrains Mono', monospace — tabular figures
한글:    'Pretendard', sans-serif (Inter 폴백)

크기:
  H1: 1.5rem (24px) — 페이지 제목
  H2: 1.125rem (18px) — 섹션
  본문: 0.875rem (14px)
  캡션: 0.75rem (12px) — 국가명, 라벨
```

## 레이아웃

```
┌─────────────────────────────────────────────┐
│  POCO Map                    [필터] [정보]  │
├─────────────────────────────┬───────────────┤
│                             │               │
│                             │  국가 패널    │
│       2D 세계지도           │  ─────────    │
│       (D3 projection)       │  국가명       │
│                             │  유물 수      │
│                             │  취득 분포    │
│                             │  유물 목록    │
│                             │               │
├─────────────────────────────┴───────────────┤
│  ◄──── 타임라인 슬라이더 ────►   1700  2025 │
└─────────────────────────────────────────────┘
```

- 지도 영역: 전체 너비의 70%
- 사이드 패널: 30%, 스크롤 가능
- 타임라인: 하단 고정
- 필터: 우상단, 토글 버튼 (아이콘 아닌 텍스트 라벨)

## 지도 스타일

- 투영법: Natural Earth 또는 Robinson (왜곡 최소)
- 국가 경계: #E0E0E0, 0.5px
- 국가 채움: #F0F0EC (지도 배경보다 살짝 어둡게)
- 바다: 배경색과 동일 (#FAFAF8) — 바다를 그리지 않는다
- 유물 원: 크기 = 유물 수 비례, 투명도 0.7
- 호버: 원 테두리 #1A1A1A, 2px

## 인터랙션

- **호버**: 툴팁 (국가명 + 유물 수). 미니멀, 그림자 없음
- **클릭**: 사이드 패널 전환. 부드러운 트랜지션 (300ms ease)
- **필터**: 토글 시 지도가 즉시 갱신. 페이드 애니메이션
- **타임라인**: 드래그 시 실시간 필터. 연도 라벨 표시
